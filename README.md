# Cloud Beginner CI/CD - GitHub Actions + Pages

This is a simple static website (HTML/CSS/JS) that deploys automatically to GitHub Pages using GitHub Actions.

## Live Website
GitHub Pages Link: (https://github.com/syzar20/cloud-beginner-cicd-pages-karim-yousef.git)

## How to run locally
1. Download the project or clone it.
2. Open `index.html` in any browser.
   - Or use VS Code extension "Live Server" and run it.
